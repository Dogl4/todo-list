# Projeto Todo list
## 📋 Descrição do projeto
Um site onde se gerencia uma lista de tarefas. Com o uso de manipulação do DOM, LocalStorage, JavaScript onde o usuário pode adicionar, mover, excluir e marcar tarefas.
